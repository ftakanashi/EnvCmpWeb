|=======================================================================================|
|  使用方法  (工具存放路径最好不要带中文)                                               |
|=======================================================================================|
|                                                                                       |
|  双击start.exe打开工具，会自动开启服务并自动打开浏览器访问服务实现工具的WEB界面化。   |
|  默认开启的服务端口是5050，如果有冲突或者需要手动指定可以在cmd中，后加上一个参数运行。|
|											|
|  工具分成两个模块，本地对比和远程对比。                                               |
| 										        |
|  本地对比的目录放在start.exe同级目录中的Compare目录下，进入工具点选本地对比功能即可对 |
|比。                                                                                   |
|  远程对比需要使用者输入两个不同目录所在的服务器的IP和用户名密码，随后工具下载对应目录 |
|到本地进行对比。                                                                       |
|                                                                                       |
|  进入对比结果展示界面后分成三个标签页，从左到右依次是只存在于左目录的文件/目录、只存在|
|右目录的文件/目录、两目录中文件名相同内容不同的文件。                                  |
|										        |
|  在第三个标签页点击文本文件可以进入详细的文本对比界面，界面上会标识出两文件内容的不同 |
|  在点击文件名前输入展示宽度和勾选只展示不同处可以控制不同的展示方式。                 |
